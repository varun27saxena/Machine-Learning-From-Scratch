# рџ“– About

This repository is my personal learning journey where I implement Machine Learning algorithms completely **from scratch** without relying on machine learning libraries for the core implementations.

The objective is to understand the mathematics, optimization techniques, and intuition behind every algorithm rather than simply using APIs.

Every implementation is built step-by-step using **Python** and **NumPy**, followed by comparisons with Scikit-Learn implementations.
